# คู่มือใช้งาน DST Manager (สำหรับผู้ใช้ทั่วไป)

คู่มือนี้สำหรับคนที่โหลดไฟล์ **`dst-manager.exe`** ไปใช้ — ไม่ต้องเขียนโค้ด ไม่ต้องลงโปรแกรมเสริม
ทำตามทีละขั้นได้เลย

> **DST Manager ทำอะไร?** ช่วยเปิด/ปิด/ดูแล server เกม Don't Starve Together ของคุณ
> สั่งงานได้ทั้งจาก **หน้าเว็บในเครื่อง** และ **บอท Discord**

---

## ✅ ก่อนเริ่ม ต้องมีอะไรบ้าง

โปรแกรมนี้แค่ "ควบคุม" server ที่มีอยู่แล้ว — มันไม่ได้ติดตั้ง/สร้าง server ให้ ดังนั้นต้องเตรียม:

1. **เครื่อง Windows** ที่ลง **DST Dedicated Server** ไว้แล้ว
   (โหลดผ่าน Steam → Library → Tools → "Don't Starve Together Dedicated Server")
2. **สร้าง cluster ไว้แล้ว** — คือโฟลเดอร์โลกที่อยู่ใน
   `Documents\Klei\DoNotStarveTogether\<ชื่อ cluster>\`
   ข้างในต้องมี `cluster.ini`, `cluster_token.txt` และโฟลเดอร์ `Master\` (และ `Caves\` ถ้ามีถ้ำ)
3. **Discord bot** ของตัวเอง (วิธีสร้างอยู่ด้านล่าง — ใช้เวลา ~3 นาที)

> ⚠️ **ต้องรันโปรแกรมบนเครื่องเดียวกับ server เกม** เพราะมันสั่งเปิดตัวเกม server ในเครื่องจริง ๆ

---

## 1) โหลดและเปิดโปรแกรม

1. โหลด `dst-manager-windows-x64.exe` จากหน้า **Releases** ของโปรเจกต์
2. สร้างโฟลเดอร์ว่าง ๆ (เช่น `C:\DST-Manager`) แล้วเอาไฟล์ไปวางไว้ในนั้น
   (โปรแกรมจะสร้างไฟล์ตั้งค่าข้าง ๆ ตัวมันเอง — อยู่ในโฟลเดอร์ว่างจะเป็นระเบียบกว่า)
3. **ดับเบิลคลิก** ไฟล์ `.exe`

### 🛡️ เจอหน้าต่างเตือนของ Windows?

ไฟล์นี้ไม่ได้เซ็น digital signature (เพราะเป็นโปรแกรมฟรี/โอเพนซอร์ส) Windows เลยเตือนไว้ก่อน:

- ขึ้น **"Windows protected your PC"** → กด **More info** → **Run anyway**
- ถ้า **แอนตี้ไวรัส** ลบ/กักไฟล์ → เป็น false positive ที่พบบ่อยกับโปรแกรมแบบนี้
  ให้ตั้ง allow/exception ให้ไฟล์ (โหลดจากหน้า Releases ทางการเท่านั้น)

เปิดสำเร็จจะมี **หน้าต่างดำ (console)** ขึ้นมาแบบนี้:

```
========================================
  Web UI:  http://127.0.0.1:8787
  Token:   a1b2c3d4e5...   ← รหัสยาว ๆ ของคุณ
========================================
```

แล้วเบราว์เซอร์จะเปิดหน้าเว็บให้เอง (ถ้าไม่เปิด พิมพ์ `http://127.0.0.1:8787` ในเบราว์เซอร์เอง)

> ❗ **อย่าปิดหน้าต่างดำ** — มันคือตัวโปรแกรมที่กำลังทำงานอยู่ ปิดเมื่อไหร่ server/บอทก็หยุด

---

## 2) ใส่ Token เข้าหน้าเว็บ

ครั้งแรกที่เข้าเว็บ มันจะถาม **Web token** → ก๊อปรหัสยาว ๆ จากบรรทัด `Token:` ในหน้าต่างดำมาวาง
(token นี้เอาไว้กันคนอื่นเข้ามาสั่งงาน เก็บไว้ไม่ต้องจำ เดี๋ยวเบราว์เซอร์จำให้)

---

## 3) สร้าง Discord bot (ทำครั้งเดียว)

1. เข้า https://discord.com/developers/applications → **New Application** → ตั้งชื่อ
2. ได้ **Application ID** มาแล้ว (อยู่หน้า General Information) = ค่า **Client ID**
3. เมนูซ้าย **Bot** → **Reset Token** → ก๊อป **token** เก็บไว้ (นี่คือ **Bot Token**)
   - ✅ **ไม่ต้องเปิด** สวิตช์ privileged intents ใด ๆ (Message Content ฯลฯ) — โปรแกรมนี้ไม่ใช้
4. เมนูซ้าย **OAuth2** → เลือก scope `bot` และ `applications.commands`
   → ในส่วน Bot Permissions ติ๊ก **Manage Channels** และ **Manage Roles**
   → ก๊อปลิงก์ด้านล่างไปเปิดในเบราว์เซอร์ เพื่อ **เชิญบอทเข้าเซิร์ฟเวอร์ Discord** ของคุณ

### หา Guild ID (= Server ID ของ Discord คุณ)

- เปิด Discord → **User Settings → Advanced → เปิด Developer Mode**
- คลิกขวาที่ไอคอนเซิร์ฟเวอร์ → **Copy Server ID** = ค่า **Guild ID**

> (ทางเลือก) อยากจำกัดว่าใครสั่งคำสั่งควบคุมได้ ให้คลิกขวาที่ "ยศ/role" → Copy Role ID
> แล้วเอาไปใส่ช่อง **Admin Role ID** — ถ้าเว้นว่าง ใครก็สั่งได้ (เหมาะตอนทดลองเท่านั้น)

---

## 4) กรอกค่าตั้งค่าในหน้าเว็บ

ในหน้าเว็บ การ์ด **"ตั้งค่า (config.json)"** กรอกช่องสำคัญ (★ = จำเป็น):

| ช่อง | ใส่อะไร |
|---|---|
| ★ **Bot Token** | token จากขั้นตอนที่ 3 |
| ★ **Client ID** | Application ID |
| ★ **Guild ID** | Server ID ของ Discord คุณ |
| ★ **Install Dir** | โฟลเดอร์ที่ลง DST server (มีโฟลเดอร์ `bin64` ข้างใน) เช่น `C:\Program Files (x86)\Steam\steamapps\common\Don't Starve Together Dedicated Server` |
| ★ **Cluster** | ชื่อโฟลเดอร์ cluster ของคุณ (เช่น `MyDediServer`) |
| **Persistent Root** | โฟลเดอร์ที่เก็บ cluster — ปกติคือ `C:\Users\<ชื่อคุณ>\Documents\Klei` |
| **Shards** | เว้นว่างได้ (โปรแกรมจะหา Master/Caves ให้เอง) |

> 💡 **Persistent Root + Cluster** ต้องชี้ให้ถูก ไม่งั้นโปรแกรมหาโลกไม่เจอ
> ตรวจง่าย ๆ: เส้นทาง `<Persistent Root>\DoNotStarveTogether\<Cluster>\cluster.ini` ต้องมีไฟล์อยู่จริง

กรอกเสร็จกด **💾 บันทึก config** — ถ้ายังกรอกไม่ครบ ระบบจะบอกว่าขาดอะไร

---

## 5) เริ่มใช้งาน

1. กดปุ่ม **▶️ รันบอท** — บอทจะ login เข้า Discord, สร้างห้องให้อัตโนมัติ, ลงทะเบียนคำสั่ง
2. กดปุ่ม **▶️ Start** (ในการ์ด "ควบคุม DST") เพื่อ **เปิด server เกม**
3. ดูสถานะ ผู้เล่น วัน/ฤดู ได้ในการ์ด "สถานะ server"

ตอนนี้สั่งงานได้ 2 ทาง:
- **หน้าเว็บนี้** — ปุ่ม Start / Stop / Restart / Save / Backup
- **ใน Discord** — พิมพ์ slash command เช่น `/status`, `/players`, `/start`, `/announce`
  (บอทสร้างห้องควบคุมที่มีปุ่มกดให้ด้วย)

## 📁 ไฟล์ที่โปรแกรมสร้าง (อยู่ข้าง ๆ ตัว .exe)

| ไฟล์ | คืออะไร |
|---|---|
| `config.json` | ค่าตั้งค่าทั้งหมด **มี token ลับอยู่ในนี้ — ห้ามส่งให้ใคร / ห้ามอัปขึ้นเน็ต** |
| `channels.json` | จำว่าบอทสร้างห้อง Discord ไหนไว้บ้าง |
| `backups/` | ไฟล์สำรองเซฟโลก (`.tar.gz`) ตอนกด Backup |

> ย้ายโปรแกรมไปเครื่องใหม่/โฟลเดอร์ใหม่ ก็ก๊อป `config.json` ไปด้วยเพื่อไม่ต้องตั้งค่าใหม่

## 🧯 ปัญหาที่เจอบ่อย

| อาการ | วิธีแก้ |
|---|---|
| หน้าเว็บ **หน้าตาเพี้ยน/ขาวโล่ง** | โปรแกรมโหลดดีไซน์จากเน็ต — ต้องต่ออินเทอร์เน็ตตอนเปิดเว็บ |
| เปิดเว็บแล้วขึ้น **"ขาดการเชื่อมต่อ"** | หน้าต่างดำ (โปรแกรม) ถูกปิดไป → เปิด `.exe` ใหม่ |
| ใส่ token แล้วเด้งว่า **token ผิด** | ใช้ค่าจากบรรทัด `Token:` ในหน้าต่างดำ (ไม่ใช่ Discord token) |
| กด **รันบอท** ไม่ได้ / ขึ้นว่ายังกรอกไม่ครบ | กรอกช่อง ★ ให้ครบแล้วกดบันทึกก่อน |
| ขึ้นแถบแดง **shard crash** | กดดู log ใน Discord ห้อง `dst-logs` หรือเปิด server_log.txt — มักเป็นปัญหาม็อด/ค่าโลก |
| บอท online แต่ **ไม่มีห้อง/คำสั่ง** | เช็คว่าเชิญบอทด้วยสิทธิ์ Manage Channels + Manage Roles แล้ว |
| port ชนกัน (เปิดได้ทีละตัว) | ปิดโปรแกรมเดิมให้หมดก่อนเปิดใหม่ |

---

## 🔒 เรื่องความปลอดภัย

- โปรแกรมเปิดหน้าเว็บไว้ที่ **เครื่องตัวเองเท่านั้น** (`127.0.0.1`) คนนอกเข้าไม่ได้
- ทุกการสั่งงานต้องมี token — เก็บ `config.json` ไว้ให้ดี
- **อย่าเปิดให้เข้าจากเครื่องอื่นแบบตรง ๆ** ถ้าจำเป็นต้องเข้าจากนอกเครื่อง ให้คนที่เข้าใจระบบ
  ตั้ง reverse proxy + HTTPS ให้
