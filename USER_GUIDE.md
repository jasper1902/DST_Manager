# คู่มือใช้งาน DST Manager (สำหรับผู้ใช้ทั่วไป)

คู่มือนี้สำหรับคนที่โหลดไฟล์ **`dst-manager.exe`** ไปใช้ — ไม่ต้องเขียนโค้ด ไม่ต้องลงโปรแกรมเสริม
ทำตามทีละขั้นได้เลย

> **DST Manager ทำอะไร?** ช่วย **ติดตั้ง / เปิด / ปิด / ดูแล** server เกม Don't Starve Together ของคุณ
> สั่งงานได้ทั้งจาก **หน้าเว็บในเครื่อง** และ **บอท Discord** (สลับภาษา ไทย/อังกฤษ ได้)

---

## ✅ ก่อนเริ่ม ต้องมีอะไรบ้าง

โปรแกรมนี้ **ดาวน์โหลดตัว server เกมให้เอง** และ **สร้างโลกใหม่ให้อัตโนมัติ** — คุณแทบไม่ต้องเตรียมอะไร
สิ่งที่ต้องมีจริง ๆ คือ:

1. **เครื่อง Windows** + ต่ออินเทอร์เน็ต (ตัว server เกมหลาย GB จะถูกโหลดมาให้)
2. **Discord bot** ของตัวเอง (วิธีสร้างอยู่ด้านล่าง — ~3 นาที)
3. **Server Token** ของ Klei (รหัสยาว ๆ ขอฟรีที่ https://accounts.klei.com → Game Servers)

> ✅ **ไม่ต้องลง DST server เองล่วงหน้า** และ **ไม่ต้องสร้างโฟลเดอร์โลกเอง** — โปรแกรมจัดให้
> (ถ้าอยากเอา **โลกเดิม** ที่มีอยู่มาใช้ ดูหัวข้อ *นำเข้าโลก (Import)* ท้ายคู่มือ)

> ⚠️ **ต้องรันโปรแกรมบนเครื่องเดียวกับที่จะให้ server เกมรัน** เพราะมันเปิดตัวเกม server ในเครื่องจริง ๆ

---

## 1) โหลดและเปิดโปรแกรม

1. โหลด `dst-manager-windows-x64.exe` จากหน้า **Releases** ของโปรเจกต์
2. สร้างโฟลเดอร์ว่าง ๆ **ที่ path ไม่มีช่องว่าง** (เช่น `C:\DST-Manager`) แล้วเอาไฟล์ไปวางไว้
   (โปรแกรมจะสร้างไฟล์/โฟลเดอร์ของมันข้าง ๆ ตัวเอง — รวมถึงตัว server เกมที่โหลดมา)
3. **ดับเบิลคลิก** ไฟล์ `.exe`

### 🛡️ เจอหน้าต่างเตือนของ Windows?

ไฟล์นี้ไม่ได้เซ็น digital signature (เพราะเป็นโปรแกรมฟรี/โอเพนซอร์ส) Windows เลยเตือนไว้ก่อน:

- ขึ้น **"Windows protected your PC"** → กด **More info** → **Run anyway**
- ถ้า **แอนตี้ไวรัส** ลบ/กักไฟล์ → เป็น false positive ที่พบบ่อย ให้ตั้ง allow/exception ให้ไฟล์
  (โหลดจากหน้า Releases ทางการเท่านั้น)

เปิดสำเร็จจะมี **หน้าต่างดำ (console)** ขึ้นมาแบบนี้:

```
========================================
  Web UI:  http://127.0.0.1:8787
  Token:   a1b2c3d4e5...   ← รหัสยาว ๆ ของคุณ
========================================
```

แล้วเบราว์เซอร์จะเปิดหน้าเว็บให้เอง (ถ้าไม่เปิด พิมพ์ `http://127.0.0.1:8787` ในเบราว์เซอร์เอง)

> ❗ **อย่าปิดหน้าต่างดำ** — มันคือตัวโปรแกรมที่กำลังทำงานอยู่ ปิดเมื่อไหร่ server/บอทก็หยุด

---

## 2) ใส่ Web Token เข้าหน้าเว็บ

ครั้งแรกที่เข้าเว็บ มันจะถาม **Web token** → ก๊อปรหัสยาว ๆ จากบรรทัด `Token:` ในหน้าต่างดำมาวาง
(token นี้กันคนอื่นเข้ามาสั่งงาน เก็บไว้ไม่ต้องจำ เบราว์เซอร์จำให้)

หน้าเว็บแบ่งเป็น 3 แท็บ: **🏠 Home** (สถานะ + ปุ่มเริ่ม), **🧙 Setup** (ตั้งค่าทีละขั้น), **⚙️ Advanced** (ของขั้นสูง)
มุมขวาบนสลับภาษา **English / ไทย** ได้

---

## 3) สร้าง Discord bot (ทำครั้งเดียว)

1. เข้า https://discord.com/developers/applications → **New Application** → ตั้งชื่อ
2. ได้ **Application ID** มาแล้ว (หน้า General Information) = ค่า **Client ID**
3. เมนูซ้าย **Bot** → **Reset Token** → ก๊อป **token** เก็บไว้ (นี่คือ **Bot Token**)
   - ✅ **ไม่ต้องเปิด** privileged intents ใด ๆ (Message Content ฯลฯ) — โปรแกรมนี้ไม่ใช้
4. เมนูซ้าย **OAuth2** → เลือก scope `bot` และ `applications.commands`
   → Bot Permissions ติ๊ก **Manage Channels** และ **Manage Roles**
   → ก๊อปลิงก์ด้านล่างไปเปิดในเบราว์เซอร์ เพื่อ **เชิญบอทเข้าเซิร์ฟเวอร์ Discord** ของคุณ

### หา Discord Server ID (= Guild ID)

- เปิด Discord → **User Settings → Advanced → เปิด Developer Mode**
- คลิกขวาที่ไอคอนเซิร์ฟเวอร์ → **Copy Server ID** = ค่า **Guild ID**

> (ทางเลือก) จำกัดว่าใครสั่งคำสั่งควบคุมได้ → คลิกขวาที่ "ยศ/role" → Copy Role ID
> เอาไปใส่ช่อง **Admin Role ID** — ถ้าเว้นว่าง ใครก็สั่งได้ (เหมาะตอนทดลองเท่านั้น)

---

## 4) ทำตามหน้า Setup ทีละขั้น (1–4)

ไปที่แท็บ **🧙 Setup** แล้วทำไล่ลงมา (แต่ละขั้นมีเครื่องหมาย ✓ บอกว่าผ่านแล้ว):

| ขั้น | ทำอะไร |
|---|---|
| **1 · ติดตั้งเกมเซิร์ฟเวอร์** | กด **⬇️ ดาวน์โหลด/อัปเดต DST server** — โปรแกรมโหลดตัว server ผ่าน SteamCMD ให้เอง (หลาย GB ใช้เวลาหลายนาที มีแถบความคืบหน้า) |
| **2 · เชื่อม Discord & ตั้งชื่อโลก** | กรอก **Bot Token / Client ID / Server (Guild) ID / Admin Role (ไม่บังคับ)** และ **ชื่อโลก (cluster)** เช่น `MyServer` แล้วกด **💾 Save** (แต่ละช่องมีคำอธิบาย + ลิงก์ "ขอได้ที่ไหน") |
| **3 · Server Token** | วาง **server token** จาก https://accounts.klei.com (ปุ่มลิงก์อยู่ในขั้นนี้) แล้วกดบันทึก — **ไม่มี token นี้ start ไม่ได้** |
| **4 · เริ่ม!** | กดปุ่มเพื่อไปหน้า Home แล้วเริ่มเซิร์ฟเวอร์ |

> 💡 **ชื่อโลกไม่ต้องมีอยู่ก่อน** — ถ้ายังไม่มี โปรแกรมจะสร้าง `cluster.ini` + โลกใหม่ให้อัตโนมัติตอน start

---

## 5) เริ่มใช้งาน (หน้า 🏠 Home)

หน้า Home จะมีปุ่มใหญ่ที่เปลี่ยนตามสถานะให้กดตามนั้นได้เลย:

- ตั้งค่ายังไม่ครบ → ปุ่ม **ไปตั้งค่า**
- พร้อมแล้ว → ปุ่มเขียว **🟢 เปิดเซิร์ฟเวอร์** (กดทีเดียว: เปิดบอท → เริ่มเกมให้ต่อเอง)
- ออนไลน์แล้ว → โชว์ 🟢 **ONLINE** + จำนวนผู้เล่น/วัน/ฤดู + ปุ่ม **⏹️ หยุด / 🔄 รีสตาร์ท / 💾 เซฟ / 🗄️ Backup**

สั่งงานได้ 2 ทาง:
- **หน้าเว็บ** — ปุ่มบนหน้า Home
- **ใน Discord** — บอทสร้างห้องควบคุมที่มี **ปุ่มกดครบ** (Start/Stop/Restart/Save/Backup/Status/Players/Mods/Logs/Config/Announce/Rollback/Regenerate) หรือพิมพ์ slash command เช่น `/status`, `/players`, `/mods`, `/announce`

---

## 🧩 ม็อด (mods)

- ดูม็อดที่โลกเปิดใช้: พิมพ์ `/mods` ใน Discord หรือดูในแท็บ Advanced (แปลงเลข workshop เป็นชื่อจริง คลิกไป Steam Workshop ได้)
- **ถ้านำโลกที่ใช้ม็อดมา หรือม็อดยังไม่ถูกโหลด:** ไปแท็บ **⚙️ Advanced → การ์ด Mods → ปุ่ม 🧩 ดาวน์โหลด/ติดตั้งม็อด**
  โปรแกรมจะอ่านรายชื่อม็อดของโลก แล้วโหลดผ่าน SteamCMD ให้ (ต้องหยุดบอทก่อน)
  *(ปกติแค่กด Start เฉย ๆ ตัวเกมก็โหลดม็อดให้เองอยู่แล้ว — ปุ่มนี้ไว้โหลดล่วงหน้า/ทำสำเนาในเครื่อง)*

## 📦 นำเข้าโลก (Import) — เผื่ออยากเอาโลกเดิมมาใช้

แท็บ **⚙️ Advanced → การ์ด Import World** (ต้องหยุดบอทก่อน):

1. เลือกที่มา: **อัปโหลดไฟล์ zip/tar.gz** ของโฟลเดอร์ cluster เดิม หรือ **ใส่ path โฟลเดอร์** ในเครื่อง
2. โหมด: **Full** (เอาม็อดมาด้วย) หรือ **No-mods** (ตัดม็อด/สคริปต์ออก ปลอดภัยกว่า), มีตัวเลือกสร้างโลกใหม่จากค่าเดิม
3. กด Import — ระบบจะ **backup ของเดิมก่อน** แล้วเขียนทับ (ถ้าพังจะกู้คืนให้)

> 🔐 ระบบ import จะรับเฉพาะไฟล์ที่ปลอดภัย (เซฟ/ตั้งค่า) และ **ไม่เอา `cluster_token.txt` ของคนอื่น** มาปน — ใส่ token ของคุณเองในขั้น Setup

## 📁 ไฟล์ที่โปรแกรมสร้าง (อยู่ข้าง ๆ ตัว .exe)

| ไฟล์/โฟลเดอร์ | คืออะไร |
|---|---|
| `config.json` | ค่าตั้งค่าทั้งหมด **มี token ลับ — ห้ามส่งให้ใคร / ห้ามอัปขึ้นเน็ต** |
| `channels.json` | จำว่าบอทสร้างห้อง Discord ไหนไว้ |
| `mods-cache.json` | จำชื่อม็อด (จากเลข workshop) กันดึงซ้ำ — ลบได้ |
| `games/` | ตัว DST server + โลก (cluster) ที่โหลด/สร้างมา (ใหญ่มาก) |
| `steamcmd/` | ตัวโหลด SteamCMD + ม็อด workshop ที่โหลดมา |
| `backups/` | ไฟล์สำรองเซฟโลก (`.tar.gz`) ตอนกด Backup |

> ย้ายโปรแกรมไปเครื่องใหม่ ก๊อป `config.json` ไปด้วยจะได้ไม่ต้องตั้งค่าใหม่ (ตัว server เกมโหลดใหม่ได้)

## 🧯 ปัญหาที่เจอบ่อย

| อาการ | วิธีแก้ |
|---|---|
| หน้าเว็บ **หน้าตาเพี้ยน/ขาวโล่ง** | ดีไซน์โหลดจากเน็ต — ต้องต่ออินเทอร์เน็ตตอนเปิดเว็บ |
| เปิดเว็บแล้วขึ้น **"ขาดการเชื่อมต่อ"** | หน้าต่างดำ (โปรแกรม) ถูกปิด → เปิด `.exe` ใหม่ |
| ใส่ token แล้วเด้งว่า **token ผิด** | ใช้ค่าจากบรรทัด `Token:` ในหน้าต่างดำ (คนละตัวกับ Discord token) |
| กด **เปิดเซิร์ฟเวอร์** ไม่ได้ | ทำขั้น Setup ให้ครบ: ติดตั้ง server, ตั้งค่า Discord+ชื่อโลก, และ **ใส่ Server Token** |
| ขึ้น **start ไม่ได้: ไม่มี cluster_token.txt** | ไปขั้น **3 · Server Token** ใส่ token จาก accounts.klei.com |
| ติดตั้ง server **ค้าง/พัง** | path โปรแกรมต้องไม่มีช่องว่าง + ต่อเน็ตอยู่ ลองกดดาวน์โหลดใหม่ |
| ขึ้นแถบแดง **shard crash** | ดู log ใน Discord ห้อง `dst-logs` — มักเป็นปัญหาม็อด/ค่าโลก |
| บอท online แต่ **ไม่มีห้อง/คำสั่ง** | เชิญบอทด้วยสิทธิ์ **Manage Channels + Manage Roles** แล้วหรือยัง |

## 🔒 เรื่องความปลอดภัย

- หน้าเว็บเปิดที่ **เครื่องตัวเองเท่านั้น** (`127.0.0.1`) คนนอกเข้าไม่ได้ และทุกการสั่งงานต้องมี token
- เก็บ `config.json` (มี token) และ `cluster_token.txt` ไว้ให้ดี **อย่าแชร์**
- **อย่าเปิดให้เข้าจากเครื่องอื่นแบบตรง ๆ** ถ้าจำเป็น ให้คนที่เข้าใจระบบตั้ง reverse proxy + HTTPS ให้
- **นำเข้าโลก (Import)** ให้ใช้กับไฟล์ที่ไว้ใจเท่านั้น — โหมด **No-mods** ปลอดภัยสุด (ไม่รันสคริปต์ของคนอื่น)
